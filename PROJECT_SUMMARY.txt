PROJECT OVERVIEW
----------------
A comprehensive, enterprise-grade Capture The Flag (CTF) web application 
designed for teaching web security and penetration testing. The application 
appears as a legitimate educational institute website but contains 34 
carefully crafted security vulnerabilities.

DELIVERABLES
------------
1. Complete Flask web application (app.py - 800+ lines)
2. 20+ HTML templates with realistic UI
3. CSS and JavaScript files
4. Comprehensive exploitation guide (120+ pages)
5. Quick reference card for instructors
6. README with installation instructions
7. Run script for easy deployment
